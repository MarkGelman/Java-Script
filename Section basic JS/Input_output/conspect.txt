Alert -- выводит сообщение пользователю ввиде всплывающего окна браузера в самом верху экрана. Это окно
         служит для выведения каких-то важных сообщений пользовател, особенно когда хотим предупредить его
         о чём то

Confirm -- это тоже всплывающее окно всамом вверху экрана, который служит для получения от пользователя 
           ответа типа: "оК" или "Отмена" на выбор. Результат будет записан в переменную ( в нашем случае
           в переменную result -- это всегда будет булевая переменная)

Prompt -- это тоже всплывающее окно всамом вверху экрана, но там пользователь может набрать ответ с 
          клавиатуры. При чём prompt -- это метод который принимает всегда два аргумента через запятую:
          1) Сам вопрос адрессованый пользователю
          2) Ответ по умолчанию, если пользователь нажмёт на клавишу "Отмена"
          3) Этот метод всегда возвращает тип "String", но если перед ним поставить знак "+", то 
             возвращаемый тип будет "number"

document.write("") -- это очень старая уже не использующаяся команда, которая непосредственно работает со страницей
                      браузера
